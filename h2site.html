Welcome, This is a Short tutorial on how to make your own ultra-learn

First, go to https://github.com/search?q= and search ultra-learn owner:pretzer25 in the searchbar


Secondly, if you have a github account, fork this repository, if not make one


now go to https://app.netlify.com/ and Log in with GitHub,

then click add site in the middle of the page, import from existing project and give permission


now select the repository that is forked from ultra-learn, your site name is basically your first letters of your site name, leave all the build settings emty, delete if something is in them


Without changing anything, click deploy and wait to get your site!


To keep your site updated, check your own github repository after each day and sync changes (this option is showed above all the code)
